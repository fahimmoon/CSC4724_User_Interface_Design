# CSC4724_User_Interface_Design

Responsive UI demo showcasing keyframes + animations in pure HTML/CSS.

## What’s inside
- Responsive hero layout (desktop split → mobile stacked)
- Responsive image grids (4 → 2 columns)
- Infinite horizontal image loop (CSS keyframes, seamless)
- Animated sections (hero/nav entrance animations)
- Reduced-motion support (`prefers-reduced-motion`)

## How to run
1. Open `index.html` in your browser.
2. Resize the window to test responsiveness.

## Project structure
- `index.html` — page markup
- `style.css` — styling, responsiveness, keyframes/animations
- `assets/` — images used by the layout

## Notes
- The infinite loop uses a duplicated image set to keep scrolling seamless.
- If your OS/browser has reduced motion enabled, animations are minimized automatically.
